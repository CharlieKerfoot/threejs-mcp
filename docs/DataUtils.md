# DataUtils

A class containing utility functions for data.

## Static Methods
- `.fromHalfFloat( val :number) : number` — Returns a single precision floating point value (FP32) from the given half
precision floating point value (FP16).
- `.toHalfFloat( val :number) : number` — Returns a half precision floating point value (FP16) from the given single
precision floating point value (FP32).
