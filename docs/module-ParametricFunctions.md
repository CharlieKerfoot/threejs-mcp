## Methods
- `.klein( v :number, u :number, target :Vector3) (inner)` — A parametric function representing the Klein bottle.
- `.mobius( u :number, t :number, target :Vector3) (inner)` — A parametric function representing a flat mobius strip.
- `.mobius3d( u :number, t :number, target :Vector3) (inner)` — A parametric function representing a volumetric mobius strip.
- `.plane( u :number, v :number, target :Vector3) (inner)` — A parametric function representing a flat plane.
