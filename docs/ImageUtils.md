# ImageUtils

A class containing utility functions for images.

## Static Methods
- `.getDataURL( image :HTMLImageElement | HTMLCanvasElement, type :string) : string` — Returns a data URI containing a representation of the given image.
- `.sRGBToLinear( image :HTMLImageElement | HTMLCanvasElement | ImageBitmap | Object) : HTMLCanvasElement | Object` — Converts the given sRGB image data to linear color space.
