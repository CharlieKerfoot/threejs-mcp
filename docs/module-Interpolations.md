# Interpolations

## Methods
- `.CatmullRom( t :number, p0 :number, p1 :number, p2 :number, p3 :number) : number` — Computes a point on a Catmull-Rom spline.
- `.CubicBezier( t :number, p0 :number, p1 :number, p2 :number, p3 :number) : number` — Computes a point on a Cubic Bezier curve.
- `.QuadraticBezier( t :number, p0 :number, p1 :number, p2 :number) : number` — Computes a point on a Quadratic Bezier curve.
