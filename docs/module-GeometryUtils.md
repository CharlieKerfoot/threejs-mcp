## Methods
- `.gosper( size :number) : Array.<number>` — Generates a Gosper curve (lying in the XY plane). Reference: https://gist.github.com/nitaku/6521802
- `.hilbert2D( center :Vector3, size :number, iterations :number, v0 :number, v1 :number, v2 :number, v3 :number) : Array.<Vector3>` — Generates 2D-Coordinates along a Hilbert curve. Based on work by: http://www.openprocessing.org/sketch/15493
- `.hilbert3D( center :Vector3, size :number, iterations :number, v0 :number, v1 :number, v2 :number, v3 :number, v4 :number, v5 :number, v6 :number, v7 :number) : Array.<Vector3>` — Generates 3D-Coordinates along a Hilbert curve. Based on work by: https://openprocessing.org/user/5654
